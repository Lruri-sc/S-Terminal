# Terminals that support non-equisupporting fonts

A Terminal that support non-equisupporting fonts terminal

## Features

- 🎨 Fully customizable (fonts, colors, backgrounds).
- 🖼️ Background image support
- 📋 Copy/paste support
- 🌈 Full color support
- 💾 User configuration in `~/chloe-term/`

## Installation
```bash
npm install
npm start
```

## Build
```bash
npm run dist:mac
```

## Configuration

Configuration file: `~/chloe-term/config.txt`

Edit and restart the app to apply changes.
